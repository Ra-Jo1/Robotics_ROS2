Overview :

This repository reflects my ongoing career transition from **Data Engineering** to **Robotics Engineering**. After several years working with data pipelines, distributed systems, data migration, I have decided to pivot toward robotics.
I’ve thought a lot about what I really want to do in the long term in my professional life, and what I wanted was to work with innovative, cutting-edge technology and integrate software into a robot that moves around in the real world. I firmly believe that certain technologies can positively improve our lives if used properly, and that humanoid robots will be among the technologies that contribute to this in the future.

Background :

As a Data Engineer, I have developed strong expertise in:

* Designing and maintaining data pipelines
* Working with large datasets and distributed systems
* Programming (primarily in Python and VBA)
* Data infrastructure
* Problem-solving and system optimization

These skills provide a solid foundation for robotics, especially in areas such as perception, data processing, and system architecture.

To support this transition, I am actively developing skills in:

* Robotics fundamentals (kinematics, dynamics, control)
* Programming for robotics (C++, Python, ROS)
* Simulation tools (Gazebo, RViz)
* Embedded systems basics
* Computer vision and sensor data processing

This repository will include projects, experiments, and learning notes related to these topics.
